# SIMTization and SIMDization of Verlet list construction for molecular dynamics simulations

## Compile and run

### @ Haswell Xeon CPU

``` bash
$ make hsw
$ ./make_list_cpu_4x1.out
```

### @ Skylake Xeon CPU

``` bash
$ make skl
$ ./make_list_cpu_8x1.out
```

### @ Knights Landing

``` bash
$ make knl
$ ./make_list_mic_8x1.out
```

### @ NVIDIA GPU

``` bash
$ make gpu
$ ./make_list_gpu_warp_unroll_smem.out
```
